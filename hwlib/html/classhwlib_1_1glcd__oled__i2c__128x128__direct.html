Detailed Description

OLED direct class for the i2c connected 128x128 Grayscale OLED based on ssd1327.

This class writes to the OLED window directly (without writing and flushing a buffer). This class contains a buffer array (size 8192) and a constant size. The class also contains several overwritten write functions due to name hiding When writing a pixel one byte is written to the OLED (containing 2-pixel information) TODO: make sure that a byte is not written twice to the OLED when writing a consecutive area
