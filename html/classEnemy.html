void Enemy::update ( )
overridevirtual

Void update function.

This function checks if the enemy is active. Then calls on the draw inverse function to remove the image of the enemy. The speed is added to the y axis of the location. If the location of the enemy is outside the screen the object is disactivated and placed on the top of screen. If the Enemy is inactive its activated.
